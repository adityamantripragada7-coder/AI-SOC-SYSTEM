# 🚀 AI-Powered SOC Threat Detection System

An end-to-end AI-based Security Operations Center (SOC) system that detects cyber threats using Machine Learning, integrates threat intelligence, calculates risk scores, and visualizes results via a dashboard.

---

## 🔥 Features

- 🤖 Machine Learning-based intrusion detection
- 🌐 IP-based threat intelligence analysis
- ⚠️ Risk scoring system
- 🚨 Automated alert generation
- 📊 Streamlit dashboard visualization
- 🔄 End-to-end pipeline from logs → prediction → alerts

---

## 🧠 Tech Stack

- Python
- Pandas
- Scikit-learn
- Joblib
- Streamlit
- Git & GitHub
  
---

## ⚙️ Setup Instructions

### 1. Clone Repository

git clone https://github.com/adityamantripragada7-coder/AI-SOC-SYSTEM.git  
cd AI-SOC-SYSTEM

---

### 2. Create Virtual Environment

python -m venv venv  
venv\Scripts\activate

---

### 3. Install Dependencies

pip install pandas scikit-learn joblib streamlit

---

### 4. Train Model

cd model  
python train.py

---

### 5. Run Main System

cd ..  
python main.py

---

### 6. Launch Dashboard

streamlit run dashboard.py

---

## 📊 Dashboard Output

- Shows IP addresses
- Predicts attack / normal
- Displays risk score
- Highlights high-risk alerts 🚨
